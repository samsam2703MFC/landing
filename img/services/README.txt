Illustrations des tuiles « Nos services ».

Déposez ici les deux fichiers fournis par le graphiste :

  delivery.png   → tuile « Livraison au bureau » (main + colis, trait continu)
  b2b.png        → tuile « Comptes B2B » (B2B dans une boucle, trait continu)

Format conseillé : PNG fond transparent, carré (~800×800 px), < 150 Ko.

Ces chemins sont déjà renseignés dans la table lp_services (colonne image_path).
Tant que les fichiers ne sont pas là, les tuiles affichent l'icône SVG de repli
automatiquement — aucune image cassée.

Pour changer le chemin : lp_services.image_path (PHPMyAdmin).
